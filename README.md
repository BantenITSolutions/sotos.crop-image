#sotos.image-crop module for angular
================
directive in angular 

crop images, put watermark and save directive in angular 

Use

  `<html>
  <body>
    <image-crop> image-out="imageOut"  crop-options="options" ng-transclude>
       <edit-crop></edit-crop>
       <view-crop></view-crop>
    </image-crop>
  `</body>`
  `</html>`
